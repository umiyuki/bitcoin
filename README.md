Sayacoinについて
========

Sayacoin（さやコイン）はBitcoin0.8.6ベースの仮想通貨です。
万が一さやかさんが仮想通貨のスタンダードになったら面白いな～と思って作りました。
さやかさんファンは集めてはいかがでしょうか。

Sayacoinの仕様
========

アルゴリズムはBitcoinと同じくSHA-256です。

 - コイン最大数　2100万枚
 - アルゴリズム　SHA-256
 - ブロック時間　5分
 - Difficulty再設定間隔　12ブロック
 - ブロック報酬半減　210000ブロック
 - 最初の報酬　50SYC
